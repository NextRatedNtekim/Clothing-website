#  Wearix – Modern Fashion E-Commerce Store

Wearix is a premium fashion e-commerce web application built with React, Vite, Tailwind CSS, Framer Motion, and Lucide React. The platform delivers a seamless shopping experience with modern UI/UX, responsive layouts, product browsing, cart management, and a complete checkout flow.

---

##  Features

###  Shopping Experience

* Browse fashion collections
* Product detail pages
* Product ratings and reviews
* Category filtering
* Product sorting
* Responsive product grid
* Dynamic product loading from API

###  Cart Management

* Add to cart
* Remove from cart
* Update product quantities
* Animated cart interactions
* Persistent shopping experience

###  Checkout System

* Order summary
* Dynamic shipping calculation
* Order confirmation
* Printable receipt generation
* Unique order numbers
* Modern checkout modal

###  Modern User Interface

* Mobile-first responsive design
* Smooth Framer Motion animations
* Premium fashion-inspired aesthetic
* Skeleton loading states
* Interactive hover effects
* Elegant typography

##  Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/wearix.git
```

Navigate into the project:

```bash
cd wearix
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

##  Build For Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

##  Environment Variables

If using a custom API:

```env
VITE_API_URL=https://your-api-url.com
```

Access inside React:

```javascript
const API_URL = import.meta.env.VITE_API_URL;
```
